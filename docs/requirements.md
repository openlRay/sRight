# sRight 需求文档

## 产品定位

sRight 是一个 macOS Finder 右键增强工具，目标是覆盖“超级右键”的主要能力，并提供更强的可配置性、动作日志、自定义脚本和条件菜单能力。

当前项目面向本机自用，不作为公开产品发布。需求文档记录完整能力边界，实际实现可以按阶段推进。

## 完整能力范围

### 1. 通用设置

- 启用或禁用 Finder 右键菜单。
- 显示或隐藏菜单图标。
- 配置菜单分组是否合并显示。
- 检测 Finder Extension 是否启用。
- 提供 Finder Extension 启用引导。
- 提供必要权限说明。
- 管理动作日志开关。
- 导入配置。
- 导出配置。
- 重置配置。
- 查看应用版本、构建信息和诊断信息。

### 2. 菜单配置

- 配置一级菜单和二级菜单。
- 支持菜单拖拽排序。
- 支持菜单启用和禁用。
- 支持菜单显示名称编辑。
- 支持菜单图标配置。
- 支持按文件类型显示菜单。
- 支持按文件数量显示菜单。
- 支持按文件或文件夹显示菜单。
- 支持按扩展名显示菜单。
- 支持按所在目录显示菜单。
- 支持危险动作二次确认。

### 3. 新建文件

- 新建空白文本文件。
- 新建 Markdown 文件。
- 新建 JSON 文件。
- 新建 XML 文件。
- 新建 shell 脚本。
- 新建 Swift 文件。
- 新建 JavaScript / TypeScript 文件。
- 新建 Word / Excel / PowerPoint 模板文件。
- 新建 WPS 模板文件。
- 支持自定义模板文件。
- 支持模板显示名称编辑。
- 支持模板拖拽排序。
- 支持模板启用和禁用。
- 支持新建后自动打开。
- 支持文件名冲突时自动追加序号。

### 4. 打开方式

- 在 Terminal 中打开。
- 在 iTerm2 中打开。
- 在 Warp 中打开。
- 在 VSCode 中打开。
- 在 Cursor 中打开。
- 在 JetBrains 系列应用中打开，例如 IDEA、WebStorm、PyCharm、GoLand、DataGrip。
- 在 Sublime Text、Typora、Obsidian、CotEditor 等编辑器中打开。
- 支持添加自定义 App。
- 支持检测 App 是否已安装。
- 支持为文件和文件夹分别配置菜单显示规则。

### 5. 文件操作

- 复制完整路径。
- 复制真实路径。
- 复制文件或文件夹名称。
- 复制父目录路径。
- 复制 shell escaped path。
- 复制 Markdown 链接。
- 移到废纸篓。
- 彻底删除。
- 根据文件名创建文件夹。
- 解散文件夹。
- 隐藏选中文件。
- 显示选中文件。
- 隐藏当前目录所有文件。
- 显示当前目录所有文件。
- 显示或隐藏文件扩展名。
- 授予选中文件写入权限。
- 查看文件信息。

### 6. 发送到与常用目录

- 复制文件到指定目录。
- 移动文件到指定目录。
- 添加常用目录。
- 删除常用目录。
- 修改常用目录显示名称。
- 修改常用目录图标。
- 支持常用目录拖拽排序。
- 支持默认目录：下载、图片、音乐、影片、文稿、桌面。
- 支持最近使用目录。
- 支持重置为默认目录。

### 7. 压缩与解压

- 压缩为 `.zip`。
- 每个选中项单独压缩为 `.zip`。
- 压缩后删除源文件。
- 解压到当前目录。
- 解压到单独目录。
- 解压后删除原压缩包。
- 支持加密压缩。
- 支持设置压缩文件名。
- 支持后续接入 `7z`。
- 支持后续修复乱码文件名。

### 8. 图片与图标工具

- 图片转 PNG。
- 图片转 JPG。
- 图片转 WebP。
- 图片转 HEIC。
- 支持批量图片转换。
- 支持配置输出质量。
- 支持输出目录配置。
- 转换为 ICNS。
- 转换为 macOS iconset。
- 设置文件或文件夹图标。
- 删除自定义图标。
- 设置图片为桌面壁纸。

### 9. 工具箱

- 文本生成二维码。
- 文件路径生成二维码。
- 计算 MD5。
- 计算 SHA1。
- 计算 SHA256。
- 翻译选中文本。
- 支持配置翻译 provider。
- 支持打开文件所在目录。
- 支持快速预览增强。
- 支持复制文件摘要信息。

### 10. 自定义脚本动作

- 添加自定义 shell 脚本动作。
- 配置脚本名称。
- 配置脚本图标。
- 配置脚本适用文件类型。
- 配置脚本是否需要二次确认。
- 支持把选中文件路径传给脚本。
- 支持脚本执行日志。
- 支持脚本执行失败提示。
- 支持禁用高风险脚本动作。

### 11. 日志与诊断

- 记录动作开始时间。
- 记录动作结束时间。
- 记录动作 ID。
- 记录选中文件数量。
- 记录执行结果。
- 记录错误信息。
- 支持日志搜索。
- 支持日志清理。
- 支持导出日志。
- 支持 CLI 诊断命令。

## 分期计划

### 第一期：工程骨架与核心链路

目标：打通从 Finder 右键菜单到动作执行的最小完整链路。

范围：

- 初始化 monorepo 工程结构。
- 初始化 Tauri 偏好设置 App。
- 初始化 Rust workspace。
- 初始化 `sright-core`。
- 初始化 `sright-cli`。
- 初始化 Swift FinderSync Extension。
- 建立配置文件目录。
- 建立结构化日志文件。
- Finder 菜单能够读取配置并显示。
- Finder 菜单能够调用 `sright-cli`。
- CLI 支持基础动作调试。

验收：

- 能在 Finder 右键菜单中看到 sRight 菜单。
- 点击菜单后能执行一个 CLI 动作。
- 偏好设置 App 能读取和写入配置。
- 日志能记录一次动作执行。

### 第二期：基础文件能力

目标：覆盖日常最高频 Finder 操作。

范围：

- 复制路径。
- 复制真实路径。
- 复制文件名。
- 复制父目录路径。
- 复制 shell escaped path。
- 移到废纸篓。
- 彻底删除。
- 根据文件名创建文件夹。
- 解散文件夹。
- 文件信息查看。
- 危险动作二次确认配置。

验收：

- 多选文件和单选文件都能正确执行。
- 危险动作默认需要确认。
- 失败时能看到清晰日志。

### 第三期：新建文件、打开方式、常用目录

目标：补齐超级右键主干工作流。

范围：

- 新建文件模板。
- 自定义模板。
- 模板排序和启用禁用。
- 打开 Terminal、iTerm2、Warp。
- 打开 VSCode、Cursor。
- 打开 JetBrains 系列应用。
- 添加自定义 App。
- 复制到常用目录。
- 移动到常用目录。
- 常用目录排序、重命名和图标配置。

验收：

- 可以通过偏好设置管理模板和常用目录。
- Finder 菜单按配置动态更新。
- 打开方式能正确处理文件和文件夹。

### 第四期：压缩解压与图片工具

目标：覆盖文件处理类增强能力。

范围：

- zip 压缩。
- 单独压缩。
- 解压到当前目录。
- 解压到单独目录。
- 压缩或解压后删除源文件配置。
- 图片转 PNG / JPG / WebP / HEIC。
- 批量图片转换。
- 转 ICNS。
- 转 macOS iconset。
- 设置和删除文件图标。

验收：

- 压缩和解压支持多选文件。
- 图片转换支持批量处理。
- 结果文件命名稳定且不覆盖原文件。

### 第五期：工具箱与高级扩展

目标：把 sRight 从功能集合提升为可扩展工具平台。

范围：

- 二维码生成。
- Hash 计算。
- 文本翻译 provider 配置。
- 自定义脚本动作。
- 条件菜单显示。
- 动作日志搜索与导出。
- 配置导入导出。
- 命令面板。

验收：

- 用户可以新增自定义脚本动作。
- 菜单能按文件类型、数量、目录等条件显示。
- 日志可以用于定位动作失败原因。
